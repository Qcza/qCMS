# qCMS
quickCMS based on MEAN with Angular2.

##Installation##
Run packages installation
>npm install

Adjust database settings in..
>config.ts

..and compile TypeScript..
>npm run tsc

..then seed database.
>npm run seed

##Starting app##
Start server
>npm start

Start server without TypeScrypt and SASS watchers
>npm run server

Default server listening port
>3000
